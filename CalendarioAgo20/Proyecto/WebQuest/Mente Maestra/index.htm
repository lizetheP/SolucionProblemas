La mente humana es uno de las cosas mas sorprendentes y desconocidas que existen. Ella puede hacer una gran cantidad de cosas: crear, resolver problemas, abstraer conceptos inexistentes en el plano físico, entender códigos y buscar patrones, etc.

El juego de Mente Maestra se trata de que uno de los jugadores debe de encontrar la combinación o clave que creo el otro jugador. El jugador que hizo el código, después de cada intento por parte del otro jugador, debe de indicarle cuántos aciertos tuvo. Por ejemplo, sí el codigo es 1234 y el jugador adivinó 1893, el programa debe decirle que tiene un número en la posición correcta y otro número en la posición incorrecta.

Tú debes de hacer un juego así. El programa debe de leer una serie de códigos (numéricos) de un archivo y escoger uno al azar. Los códigos deben de tener una longitud determinada, dependiendo del nivel de dificultad. El programa, después de cada intento por parte del usuario debe de decirle cuantas posiciones tuvo correctas y cuantos números acertó, pero que no puso en el lugar correcto. Ejemplo:

    Código: 1235
    Intentos Código #Posiciones correctas # Correctos en posiciones incorrectas
    1 1983 1 1
    2 1432 2 1
    3 1736 2 0
    4 1532 2 2
    5 1235 4 0
    GANASTE!!

    Sí el jugador no hubiera acertado el código en el número predeterminadode intentos, se revela el código y se le avisa que perdió.

    Consideraciones finales:

    - El usuario tendrá la opción del nivel de dificultad.

    - El usuario también podrá escoger si quiere jugar de nuevo una vez que gane o pierda.

    - Puedes hacer los códigos solamente numéricos, pero si quieres aumentar la dificultad puedes usar letras también.
